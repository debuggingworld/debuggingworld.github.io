1. иҙҹиҪҪеқҮиЎЎ

иҙҹиҪҪеқҮиЎЎпјҲLoad BalanceпјүпјҢе…¶еҗ«д№үе°ұжҳҜжҢҮе°ҶиҙҹиҪҪпјҲе·ҘдҪңд»»еҠЎпјүиҝӣиЎҢе№іиЎЎгҖҒеҲҶж‘ҠеҲ°еӨҡдёӘж“ҚдҪңеҚ•е…ғдёҠиҝӣиЎҢиҝҗиЎҢпјҢдҫӢеҰӮ FTP жңҚеҠЎеҷЁгҖҒWeb жңҚеҠЎеҷЁгҖҒдјҒдёҡж ёеҝғеә”з”ЁжңҚеҠЎеҷЁе’Ңе…¶е®ғдё»иҰҒд»»еҠЎжңҚеҠЎеҷЁзӯүпјҢд»ҺиҖҢеҚҸеҗҢе®ҢжҲҗе·ҘдҪңд»»еҠЎгҖӮ

зӣ®еүҚдё»жөҒзҡ„иҙҹиҪҪеқҮиЎЎж–№жЎҲеҲҶдёәд»ҘдёӢдёӨз§Қпјҡ

  • йӣҶдёӯејҸиҙҹиҪҪеқҮиЎЎпјҢеңЁж¶Ҳиҙ№иҖ…е’ҢжңҚеҠЎжҸҗдҫӣж–№дёӯй—ҙдҪҝз”ЁзӢ¬з«Ӣзҡ„д»ЈзҗҶж–№ејҸиҝӣиЎҢиҙҹиҪҪпјҢжңүзЎ¬д»¶зҡ„пјҲжҜ”еҰӮ F5пјүпјҢд№ҹжңүиҪҜд»¶зҡ„пјҲжҜ”еҰӮ NginxпјүгҖӮ
  • е®ўжҲ·з«Ҝж №жҚ®иҮӘе·ұзҡ„иҜ·жұӮжғ…еҶөеҒҡиҙҹиҪҪеқҮиЎЎпјҢRibbon е°ұеұһдәҺе®ўжҲ·з«ҜиҮӘе·ұеҒҡиҙҹиҪҪеқҮиЎЎгҖӮ

1.1 е®ўжҲ·з«Ҝзҡ„иҙҹиҪҪеқҮиЎЎ

дҫӢеҰӮ spring cloud дёӯзҡ„ ribbonпјҢе®ўжҲ·з«ҜдјҡжңүдёҖдёӘжңҚеҠЎеҷЁең°еқҖеҲ—иЎЁпјҢеңЁеҸ‘йҖҒиҜ·жұӮеүҚйҖҡиҝҮиҙҹиҪҪеқҮиЎЎз®—жі•йҖүжӢ©дёҖдёӘжңҚеҠЎеҷЁпјҢз„¶еҗҺиҝӣиЎҢи®ҝй—®пјҢиҝҷжҳҜе®ўжҲ·з«ҜиҙҹиҪҪеқҮиЎЎпјӣеҚіеңЁе®ўжҲ·з«Ҝе°ұиҝӣиЎҢиҙҹиҪҪеқҮиЎЎз®—жі•еҲҶй…ҚгҖӮ

1.2 жңҚеҠЎз«Ҝзҡ„иҙҹиҪҪеқҮиЎЎ

дҫӢеҰӮ NginxпјҢйҖҡиҝҮ Nginx иҝӣиЎҢиҙҹиҪҪеқҮиЎЎпјҢе…ҲеҸ‘йҖҒиҜ·жұӮпјҢз„¶еҗҺйҖҡиҝҮиҙҹиҪҪеқҮиЎЎз®—жі•еңЁеӨҡдёӘжңҚеҠЎеҷЁд№Ӣй—ҙйҖүжӢ©дёҖдёӘиҝӣиЎҢи®ҝй—®гҖӮеҚіеңЁжңҚеҠЎеҷЁз«ҜеҶҚиҝӣиЎҢиҙҹиҪҪеқҮиЎЎз®—жі•еҲҶй…ҚгҖӮ

1.3 еёёи§ҒиҙҹиҪҪеқҮиЎЎз®—жі•

  • йҡҸжңәпјҡйҖҡиҝҮйҡҸжңәйҖүжӢ©жңҚеҠЎиҝӣиЎҢжү§иЎҢпјҢдёҖиҲ¬иҝҷз§Қж–№ејҸдҪҝз”Ёиҫғе°‘;
  • иҪ®и®ӯпјҡиҙҹиҪҪеқҮиЎЎй»ҳи®Өе®һзҺ°ж–№ејҸпјҢиҜ·жұӮжқҘд№ӢеҗҺжҺ’йҳҹеӨ„зҗҶ;
  • еҠ жқғиҪ®и®ӯпјҡйҖҡиҝҮеҜ№жңҚеҠЎеҷЁжҖ§иғҪзҡ„еҲҶеһӢпјҢз»ҷй«ҳй…ҚзҪ®пјҢдҪҺиҙҹиҪҪзҡ„жңҚеҠЎеҷЁеҲҶй…Қжӣҙй«ҳзҡ„жқғйҮҚпјҢеқҮиЎЎеҗ„дёӘжңҚеҠЎеҷЁзҡ„еҺӢеҠӣ;
  • ең°еқҖ hashпјҡйҖҡиҝҮе®ўжҲ·з«ҜиҜ·жұӮзҡ„ең°еқҖзҡ„hashеҖјеҸ–жЁЎжҳ е°„иҝӣиЎҢжңҚеҠЎеҷЁи°ғеәҰгҖӮ
  • жңҖе°ҸиҝһжҺҘж•°пјҡеҚідҪҝиҜ·жұӮеқҮиЎЎдәҶпјҢеҺӢеҠӣдёҚдёҖе®ҡдјҡеқҮиЎЎпјҢжңҖе°ҸиҝһжҺҘж•°жі•е°ұжҳҜж №жҚ®жңҚеҠЎеҷЁзҡ„жғ…еҶөпјҢжҜ”еҰӮиҜ·жұӮз§ҜеҺӢж•°зӯүеҸӮж•°пјҢе°ҶиҜ·жұӮеҲҶй…ҚеҲ°еҪ“еүҚеҺӢеҠӣжңҖе°Ҹзҡ„жңҚеҠЎеҷЁдёҠгҖӮ

2. д»Җд№ҲжҳҜ Ribbon

Spring Cloud Ribbon жҳҜеҹәдәҺ Netflix Ribbon е®һзҺ°зҡ„дёҖеҘ—е®ўжҲ·з«Ҝзҡ„иҙҹиҪҪеқҮиЎЎе·Ҙе…·пјҢRibbon е®ўжҲ·з«Ҝз»„д»¶жҸҗдҫӣдёҖзі»еҲ—зҡ„е®Ңе–„зҡ„й…ҚзҪ®пјҢеҰӮи¶…ж—¶пјҢйҮҚиҜ•зӯүгҖӮйҖҡиҝҮ Load Balancer иҺ·еҸ–еҲ°жңҚеҠЎжҸҗдҫӣзҡ„жүҖжңүжңәеҷЁе®һдҫӢпјҢRibbon дјҡиҮӘеҠЁеҹәдәҺжҹҗз§Қи§„еҲҷпјҲиҪ®иҜўпјҢйҡҸжңәпјүеҺ»и°ғз”ЁиҝҷдәӣжңҚеҠЎгҖӮRibbon д№ҹеҸҜд»Ҙе®һзҺ°жҲ‘д»¬иҮӘе·ұзҡ„иҙҹиҪҪеқҮиЎЎз®—жі•гҖӮ

2.3 Ribbon жү©еұ•еҠҹиғҪ

1. Ribbon зӣёе…іжҺҘеҸЈ

еҸӮиҖғпјҡ org.springframework.cloud.netflix.ribbon.RibbonClientConfiguration

IRuleпјҡRibbon зҡ„иҙҹиҪҪеқҮиЎЎзӯ–з•ҘпјҢй»ҳи®ӨйҮҮз”Ё ZoneAvoidanceRule е®һзҺ°пјҢиҜҘзӯ–з•ҘиғҪеӨҹеңЁеӨҡеҢәеҹҹзҺҜеўғдёӢйҖүеҮәжңҖдҪіеҢәеҹҹзҡ„е®һдҫӢиҝӣиЎҢи®ҝй—®гҖӮ

ILoadBalancerпјҡиҙҹиҪҪеқҮиЎЎеҷЁпјҢй»ҳи®ӨйҮҮз”Ё ZoneAwareLoadBalancer е®һзҺ°пјҢе®ғе…·еӨҮдәҶеҢәеҹҹж„ҹзҹҘзҡ„иғҪеҠӣгҖӮ

IClientConfigпјҡRibbon зҡ„е®ўжҲ·з«Ҝй…ҚзҪ®пјҢй»ҳи®ӨйҮҮз”Ё DefaultClientConfigImpl е®һзҺ°гҖӮ

IPingпјҡRibbon зҡ„е®һдҫӢжЈҖжҹҘзӯ–з•ҘпјҢй»ҳи®ӨйҮҮз”Ё DummyPing е®һзҺ°пјҢиҜҘжЈҖжҹҘзӯ–з•ҘжҳҜдёҖдёӘзү№ж®Ҡзҡ„е®һзҺ°пјҢе®һйҷ…дёҠе®ғе№¶дёҚдјҡжЈҖжҹҘе®һдҫӢжҳҜеҗҰеҸҜз”ЁпјҢиҖҢжҳҜе§Ӣз»Ҳиҝ”еӣһt rueпјҢй»ҳи®Өи®ӨдёәжүҖжңүжңҚеҠЎе®һдҫӢйғҪжҳҜеҸҜз”Ёзҡ„гҖӮ

ServerListпјҡжңҚеҠЎе®һдҫӢжё…еҚ•зҡ„з»ҙжҠӨжңәеҲ¶пјҢй»ҳи®ӨйҮҮз”Ё ConfigurationBasedServerList е®һзҺ°гҖӮ

ServerListFilterпјҡжңҚеҠЎе®һдҫӢжё…еҚ•иҝҮж»ӨжңәеҲ¶пјҢй»ҳи®ӨйҮҮ ZonePreferenceServerListFilterпјҢиҜҘзӯ–з•ҘиғҪеӨҹдјҳе…ҲиҝҮж»ӨеҮәдёҺиҜ·жұӮж–№еӨ„дәҺеҗҢеҢәеҹҹзҡ„жңҚеҠЎе®һдҫӢгҖӮ

2. Ribbon иҙҹиҪҪеқҮиЎЎзӯ–з•Ҙ

  1. RandomRuleпјҡ йҡҸжңәйҖүжӢ©дёҖдёӘ ServerгҖӮ
  2. RetryRuleпјҡ еҜ№йҖүе®ҡзҡ„иҙҹиҪҪеқҮиЎЎзӯ–з•ҘжңәдёҠйҮҚиҜ•жңәеҲ¶пјҢеңЁдёҖдёӘй…ҚзҪ®ж—¶й—ҙж®өеҶ…еҪ“йҖүжӢ© Server дёҚжҲҗеҠҹпјҢеҲҷдёҖзӣҙе°қиҜ•дҪҝз”Ё subRule зҡ„ж–№ејҸйҖүжӢ©дёҖдёӘеҸҜз”Ёзҡ„ serverгҖӮ
  3. RoundRobinRuleпјҡ иҪ®иҜўйҖүжӢ©пјҢ иҪ®иҜў indexпјҢйҖүжӢ© index еҜ№еә”дҪҚзҪ®зҡ„ ServerгҖӮ
  4. AvailabilityFilteringRuleпјҡ иҝҮж»ӨжҺүдёҖзӣҙиҝһжҺҘеӨұиҙҘзҡ„иў«ж Үи®°дёә circuit tripped зҡ„еҗҺз«Ҝ ServerпјҢе№¶иҝҮж»ӨжҺүйӮЈдәӣй«ҳе№¶еҸ‘зҡ„еҗҺз«Ҝ Server жҲ–иҖ…дҪҝз”ЁдёҖдёӘAvailabilityPredicate жқҘеҢ…еҗ«иҝҮж»Ө server зҡ„йҖ»иҫ‘пјҢе…¶е®һе°ұжҳҜжЈҖжҹҘ status йҮҢи®°еҪ•зҡ„еҗ„дёӘ Server зҡ„иҝҗиЎҢзҠ¶жҖҒгҖӮ
  5. BestAvailableRuleпјҡ йҖүжӢ©дёҖдёӘжңҖе°Ҹзҡ„е№¶еҸ‘иҜ·жұӮзҡ„ ServerпјҢйҖҗдёӘиҖғеҜҹ ServerпјҢеҰӮжһң Serve rиў« tripped дәҶпјҢеҲҷи·іиҝҮгҖӮ
  6. WeightedResponseTimeRuleпјҡ ж №жҚ®е“Қеә”ж—¶й—ҙеҠ жқғпјҢе“Қеә”ж—¶й—ҙи¶Ҡй•ҝпјҢжқғйҮҚи¶Ҡе°ҸпјҢиў«йҖүдёӯзҡ„еҸҜиғҪжҖ§и¶ҠдҪҺгҖӮ
  7. ZoneAvoidanceRuleпјҡ й»ҳи®Өзҡ„иҙҹиҪҪеқҮиЎЎзӯ–з•ҘпјҢеҚіеӨҚеҗҲеҲӨж–ӯ Server жүҖеңЁеҢәеҹҹе’Ң Server зҡ„еҸҜз”ЁжҖ§йҖүжӢ© ServerпјҢеңЁжІЎжңүеҢәеҹҹзҡ„зҺҜеўғдёӢпјҢзұ»дјјдәҺиҪ®иҜў гҖӮ
  8. NacosRule: дјҳе…Ҳи°ғз”ЁеҗҢдёҖйӣҶзҫӨзҡ„е®һдҫӢпјҢеҹәдәҺйҡҸжңәжқғйҮҚгҖӮ

RestTemplate жҳҜ Spring жҸҗдҫӣзҡ„з”ЁдәҺи®ҝй—® Rest жңҚеҠЎзҡ„е®ўжҲ·з«ҜпјҢRestTemplate жҸҗдҫӣдәҶеӨҡз§ҚдҫҝжҚ·и®ҝй—®иҝңзЁӢ Http жңҚеҠЎзҡ„ж–№жі•пјҢиғҪеӨҹеӨ§еӨ§жҸҗй«ҳе®ўжҲ·з«Ҝзҡ„зј–еҶҷж•ҲзҺҮгҖӮй»ҳи®Өжғ…еҶөдёӢпјҢRestTemplate й»ҳи®Өдҫқиө– jdk зҡ„ HTTP иҝһжҺҘе·Ҙе…·гҖӮ

WebClientпјҡ

WebClient жҳҜд»Һ Spring WebFlux 5.0 зүҲжң¬ејҖе§ӢжҸҗдҫӣзҡ„дёҖдёӘйқһйҳ»еЎһзҡ„еҹәдәҺе“Қеә”ејҸзј–зЁӢзҡ„иҝӣиЎҢ Http иҜ·жұӮзҡ„е®ўжҲ·з«Ҝе·Ҙе…·гҖӮе®ғзҡ„е“Қеә”ејҸзј–зЁӢзҡ„еҹәдәҺ Reactor зҡ„гҖӮWebClient дёӯжҸҗдҫӣдәҶж ҮеҮҶ Http иҜ·жұӮж–№ејҸеҜ№еә”зҡ„ getгҖҒpostгҖҒputгҖҒdelete зӯүж–№жі•пјҢеҸҜд»Ҙз”ЁжқҘеҸ‘иө·зӣёеә”зҡ„иҜ·жұӮгҖӮ
