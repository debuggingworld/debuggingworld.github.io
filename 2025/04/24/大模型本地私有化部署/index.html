1. transformers

1.1模型下载

from modelscope import snapshot_download
snapshot_download(model_id="Qwen/Qwen2.5-0.5B-Instruct",cache_dir="/root/autodl-tmp/models")

1.2 模型调用

# 使用transformer加载qwen模型
from transformers import AutoModelForCausalLM, AutoTokenizer
import torch

DEVICE = "cuda"

# 加载本地模型路径为该模型配置文件所在的根目录
model_dir = "/root/autodl-tmp/models/Qwen/Qwen2.5-0.5B-Instruct"

# 使用transformer加载模型
model = AutoModelForCausalLM.from_pretrained(model_dir, torch_dtype="auto", device_map="auto")
tokenizer = AutoTokenizer.from_pretrained(model_dir)

# 调用模型
# 定义提示词
prompt = "你好，请介绍下你自己。"
# 将提示词封装为message
message = [{"role": "system", "content": "You are a helpful assistant system"}, {"role": "user", "content": prompt}]
# 使用分词器的apply_chat_template()方法将上面定义的消息列表进行转换;tokenize=False表示此时不进行令牌化
text = tokenizer.apply_chat_template(message, tokenize=False, add_generation_prompt=True)
print("text:\n",text)

# 将处理后的文本令牌化并转换为模型的输入张量
model_inputs = tokenizer([text], return_tensors="pt").to(DEVICE)
print("model_inputs:\n",model_inputs)

with torch.inference_mode():
    # 将数据输入模型得到输出
    response = model.generate(model_inputs.input_ids, max_new_tokens=512)
    print(response)
    # 对输出的内容进行解码还原
    response = tokenizer.batch_decode(response, skip_special_tokens=True)
    print(response)

2. Ollama

2.1 安装

官网地址

ollama命令：

Usage:
  ollama [flags]
  ollama [command]

Available Commands:
  serve       Start ollama
  create      Create a model from a Modelfile
  show        Show information for a model
  run         Run a model
  stop        Stop a running model
  pull        Pull a model from a registry
  push        Push a model to a registry
  list        List models
  ps          List running models
  cp          Copy a model
  rm          Remove a model
  help        Help about any command

Flags:
  -h, --help      help for ollama
  -v, --version   Show version information

2.2 启动模型

启动模型：

ollama run deepseek-r1:7b

测试：

# 使用openai的API风格调用ollama
from openai import OpenAI

client = OpenAI(base_url="http://localhost:11434/v1/", api_key="xxxx")

chat_completion = client.chat.completions.create(
    messages=[{"role": "user", "content": "你好，请介绍下你自己。"}], model="deepseek-r1:7b"
)
print(chat_completion.choices[0])

3. vllm

中文文档

3.1 安装

pip install vllm

3.2 启动模型

vllm serve /root/autodl-tmp/models/deepseek-ai/DeepSeek-R1-Distill-Qwen-7B --max-model-len 4096 --served-model-name DeepSeek-R1-Distill-Qwen-7B

4. LMDeploy

官网，开发者平台

4.1 安装

文档

pip install lmdeploy

4.2 启动模型

lmdeploy serve api_server -h

启动参数查询。

lmdeploy serve api_server /root/autodl-tmp/models/deepseek-ai/DeepSeek-R1-Distill-Qwen-7B --model-name DeepSeek-R1-Distill-Qwen-7B  --session-len 4096   --max-batch-size 32
