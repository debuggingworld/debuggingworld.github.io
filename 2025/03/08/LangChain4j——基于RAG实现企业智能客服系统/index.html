LangChain4j——基于RAG实现企业智能客服系统

1. 整理数据

首先，我们需要把数据整理成文档，可以是 txt、pdf、xlsx、markdown 等格式都可以。这里我们将美团外卖 - 常见问题网页中的问题和答案转成 txt 文件，【传送门】

2. 导入知识库

2.1 加载并解析文件

DocumentParser documentParser = new TextDocumentParser();
Document document = documentParser.parse(Main.class.getClassLoader().getResourceAsStream("meituan-qa.txt"));

2.2 切分文件

需要把文件中的内容切分成问答对。

自定义分割方法（按空行分割，即两个换行）：

public class CustomerServiceDocumentSplitter implements DocumentSplitter {
    @Override
    public List<TextSegment> split(Document document) {
        List<TextSegment> segments = new ArrayList<>();

        String[] parts = split(document.text());
        for (String part : parts) {
            segments.add(TextSegment.from(part));
        }
        return segments;
    }

    public String[] split(String text) {
        return text.split("\\s*\\R\\s*\\R\\s*");
    }
}

切分 Document 对象：

DocumentSplitter splitter = new CustomerServiceDocumentSplitter() ;
List<TextSegment> segments = splitter.split(document);

2.3 文本向量化

EmbeddingModel embeddingModel = ZhipuChatModelUtil.getZhipuEmbeddingModel();
List<Embedding> embeddings = embeddingModel.embedAll(segments).content();

2.4 向量存储

得到向量后，就需要把文本和向量之间的映射关系存储下来，使用 CustomerServiceAgent 在回答问题时，能够根据向量相似度找到和用户问题相似的知识库问题。

EmbeddingStore<TextSegment> embeddingStore = RedisEmbeddingStore.builder()
        .host("127.0.0.1").port(16379).dimension(1024).build();
embeddingStore.addAll(embeddings, segments);

3. ContentRetriever

ContentRetriever contentRetriever = EmbeddingStoreContentRetriever.builder()
        .embeddingStore(embeddingStore).embeddingModel(embeddingModel)
        .maxResults(5) // 最相似的5个结果
        .minScore(0.8) // 只找相似度在0.8以上的内容
        .build();

提问：

Query query = new Query("余额提现什么时候到账？");
List<Content> retrieve = contentRetriever.retrieve(query);

4.整合大模型

当我们能根据用户问题匹配到原始问题和答案后，该如何将问题的答案返回给用户呢？比如今天是 2025 年 3 月 8 号，用户问“今天的余额提现，最晚什么时候能到账？”，作为智能客服系统，能不能直接告诉客户具体的日期呢，而不是只返回一个“1-7个工作日内可退回您的支付账户”让客户来算日期，实现这个功能就可以结合大模型来实现了。

4.1 定义 Agent

public interface CustomerServiceAgent {

    String answer(String question);

    static CustomerServiceAgent getCustomerServiceAgent() {
        EmbeddingModel embeddingModel = ZhipuChatModelUtil.getZhipuEmbeddingModel();
        ChatLanguageModel chatModel = ZhipuChatModelUtil.getZhipuChatModel();
        EmbeddingStore<TextSegment> embeddingStore = RedisEmbeddingStore.builder()
                .host("127.0.0.1").port(16379).dimension(1024).build();
        ChatMemory chatMemory = MessageWindowChatMemory.withMaxMessages(10);

        ContentRetriever contentRetriever = buildContentRetriever(embeddingStore, embeddingModel);
        return AiServices.builder(CustomerServiceAgent.class).chatLanguageModel(chatModel)
                .contentRetriever(contentRetriever).chatMemory(chatMemory).build();
    }

    private static ContentRetriever buildContentRetriever(EmbeddingStore<TextSegment> embeddingStore, EmbeddingModel embeddingModel) {
        return EmbeddingStoreContentRetriever.builder()
                .embeddingStore(embeddingStore).embeddingModel(embeddingModel)
                .maxResults(5) // 最相似的5个结果
                .minScore(0.8) // 只找相似度在0.8以上的内容
                .build();
    }
}

使用：

CustomerServiceAgent serviceAgent = CustomerServiceAgent.getCustomerServiceAgent();
String answer = serviceAgent.answer("余额提现什么时候到账？");
System.out.println(answer);

结果：

余额提现通常在1-7个工作日内到账，但具体到账时间可能会因银行处理延迟而有所不同，请以账户实际到账时间为准。

没有达到我们想要的效果，我们可以这么问：

String answer = serviceAgent.answer("今天的余额提现，最晚哪天能到账？给我具体的日期");

结果：

根据您提供的信息，如果今天进行余额提现，最晚在1-7个工作日内可以到账。由于工作日不包括周末和公共假日，因此需要具体确定从今天开始计算的工作日天数。

例如，如果今天是周一，那么最晚到账的日期将是周五。如果今天是周二，那么最晚到账的日期将是下周的周二。

请您根据实际的工作日计算，从今天起算的第7个工作日即为最晚到账日期。如果您需要具体的日期，请提供今天的具体日期，我可以帮您计算出最晚到账的日期。

我们发现虽然整合了大模型，但是大模型似乎对今天是不是周末、是不是工作日这些不太智能，那这就需要利用 langchain4j 的 Tools 技术了。

4.2 整合tools

public class DateCalculator {
    @Tool("计算指定天数后的具体日期")
    String date(Integer days) {
        return LocalDate.now().plusDays(days).toString();
    }
}

绑定到 AiService 中:

return AiServices.builder(CustomerServiceAgent.class).chatLanguageModel(chatModel).tools(new DateCalculator())
        .contentRetriever(contentRetriever).chatMemory(chatMemory).build();

重新直接结果：

根据您提供的余额提现信息，一般来说1-7个工作日内可退回您的支付账户，但具体到账时间会因银行处理可能有延迟而有所不同。根据计算，您今天的余额提现最晚在2025年3月15日到账。
