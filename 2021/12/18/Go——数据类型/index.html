1. 变量

1.1 变量的声明

  1. 指定变量类型，如果没有初始化，则变量默认为零值。
  2. 根据值自行判定变量类型
  3. 简短形式，使用 := 赋值操作符（只能声明局部变量）

【注】局部变量声明后未使用，编译报错。

申明变量没有初始化就为零值

  • 数值类型为 0
  • 布尔类型为 false
  • 字符串为 “”（空字符串）
  • 复杂类型为 nil（指针、切片、map 等）

2. 常量

常量的声明和变量声明非常类似，只是把 var 换成了 const，常量在定义的时候必须赋值。

const同时声明多个常量时，如果省略了值则表示和上面一行的值相同。

2.1 iota

iota 是 go 语言的常量计数器，只能在常量的表达式中使用。

iota 在 const 关键字出现时将被重置为 0。const 中每新增一行常量声明将使 iota 计数一次( iota 可理解为 const 语句块中的行索引)。 使用 iota 能简化定义，在定义枚举时很有用。

3. 数据类型

3.1 基本数据类型

整型
类型 有无符号 占用存储空间 表示范围
uint8 无符号 1字节 0 - 255(2^8^ -1)
uint16 无符号 21字节 0 - 65535(2^16^ -1)
uint32 无符号 4字节 0 到 4294967295(2^32^ -1)
uint64 无符号 8字节 0 到 18446744073709551615(2^64^ -1)
int8 有符号 1字节 -128 到 127(-2^7^ ~ 2^7^-1)
int16 有符号 2字节 -32768 到 32767(-2^15^ ~ 2^15^-1)
int32 有符号 4字节 -2147483648 到 2147483647(-2^31^ ~ 2^31^-1)
int64 有符号 8字节 -2^63^ ~ 2^63^-1

其他整数类型：

类型 有无符号 占用存储空间 表示范围
int 有符号 32位机占4字节，64位机占8字节 (-2^31^ ~ 2^31^-1) \ ( -2^63^ ~ 2^63^-1)
uint 无符号 32位机占4字节，64位机占8字节 (0~2^32^ -1) \ ( 0 ~ 2^64^-1)
rune 有符号 等价 int32 -2^31^ ~ 2^31^-1
byte 无符号 等价 uint8 0 - 255
浮点型
类型 占用存储空间 表示范围
float32 4字节 -3.403E38 ~ 3.403E38
float64 8字节 -1.798E308 ~ 1.798E308
complex64 8字节 32 位实数和虚数
complex128 16字节 64 位实数和虚数

浮点类型存储：符号位+指数位+尾数位，所以可能会出现精度损失

数据类型转换
数值类型之间转换

Go 在不同类型的变量之间赋值时需要显式转换，并且只有显式转换(强制转换)。

语法：T(v)
T : 数据类型
v : 需要转换的变量
将值 v 转换为类型 T

数值类型转换为 string

方式1：fmt.Sprintf(“%参数”,表达式)
方式2：使用 strconv 包的函数
参数参考：https://studygolang.com/pkgdoc

string 类型转数值类型

方式：使用 strconv 包的函数

注意：string向基本数据类型转换的时候，一定要确保 string 类型能够转成有效的数据类型，否则最后得到的结果就是按照对应类型的默认值输出
