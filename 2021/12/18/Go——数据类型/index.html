Go——变量、常量与数据类型

1. 变量

1.1 变量的声明

  1. 指定变量类型，如果没有初始化，则变量默认为零值。
  2. 根据值自行判定变量类型
  3. 简短形式，使用 := 赋值操作符（只能声明局部变量）

【注】局部变量声明后未使用，编译报错。 >申明变量没有初始化就为零值 > + 数值类型为 0 > + 布尔类型为 false > + 字符串为 ""（空字符串） > + 复杂类型为 nil（指针、切片、map 等）

func main() {
	var name string = "张三"
	fmt.Println(name)

	var flag = true
	fmt.Println(flag)

	age := 18
	fmt.Println(age)

}

2. 常量

常量的声明和变量声明非常类似，只是把 var 换成了 const，常量在定义的时候必须赋值。

const同时声明多个常量时，如果省略了值则表示和上面一行的值相同。

const pi = 3.1415
const e = 2.7182

const (
	a = 1
	b = 2
)

const (
	n1 = 100
	n2
	n3
)

fmt.Println(n3) // 100

2.1 iota

iota 是 go 语言的常量计数器，只能在常量的表达式中使用。

iota 在 const 关键字出现时将被重置为 0。const 中每新增一行常量声明将使 iota 计数一次( iota 可理解为 const 语句块中的行索引)。 使用 iota 能简化定义，在定义枚举时很有用。

const (
	n1 = iota // 0
	n2        // 1
	n3        // 2
	n4        // 3
)

// 使用_跳过某些值
const (
	n5 = iota // 0
	n6        // 1
	_
	n7 // 3
)

// iota声明中间插队
const (
	n8  = iota // 0
	n9  = 100  // 100
	n10 = iota // 2
	n11        // 3
)

// 多个 iota 定义在一行
const (
	a, b = iota + 1, iota + 2 // 1,2
	c, d                      // 2,3
	e, f                      // 3,4
)

3. 数据类型

3.1 基本数据类型

整型
类型 有无符号 占用存储空间 表示范围
uint8 无符号 1字节 0 - 255(28 -1)
uint16 无符号 21字节 0 - 65535(216 -1)
uint32 无符号 4字节 0 到 4294967295(232 -1)
uint64 无符号 8字节 0 到 18446744073709551615(264 -1)
int8 有符号 1字节 -128 到 127(-27 ~ 27-1)
int16 有符号 2字节 -32768 到 32767(-215 ~ 215-1)
int32 有符号 4字节 -2147483648 到 2147483647(-231 ~ 231-1)
int64 有符号 8字节 -263 ~ 263-1

其他整数类型：

类型 有无符号 占用存储空间 表示范围
int 有符号 32位机占4字节，64位机占8字节 (-231 ~ 231-1)  ( -263 ~ 263-1)
uint 无符号 32位机占4字节，64位机占8字节 (0~232 -1)  ( 0 ~ 264-1)
rune 有符号 等价 int32 -231 ~ 231-1
byte 无符号 等价 uint8 0 - 255 
func main() {

	// 整型默认类型
	var num1 = 1
	// 打印数据类型、内存占用大小  （64 位机)
	fmt.Printf("num1 数据类型：%T，内存占用空间：%d", num1, unsafe.Sizeof(num1))
	// num1 数据类型：int，内存占用空间：8

	var num2 int8 = 128
	// 编译报错：constant 128 overflows int8
	fmt.Println(num2)
	
}
浮点型
类型 占用存储空间 表示范围
float32 4字节 -3.403E38 ~ 3.403E38
float64 8字节 -1.798E308 ~ 1.798E308
complex64 8字节 32 位实数和虚数
complex128 16字节 64 位实数和虚数

浮点类型存储：符号位+指数位+尾数位，所以可能会出现精度损失

func main() {

	//定义浮点类型的数据：
	var num1 float32 = 3.14
	fmt.Println(num1)
	//浮点数可以用十进制表示形式，也可以用科学计数法表示形式  E 大写小写都可以的
	var num2 float32 = -314E-2
	fmt.Println(num2)

	//浮点数可能会有精度的损失，所以通常情况下，建议你使用：float64
	var num3 float32 = 256.000000916
	fmt.Println(num3)  // 256
	var num4 float64 = 256.000000916
	fmt.Println(num4)  // 256.000000916

	// 默认的浮点类型为：float64
	var num5 = 3.17
	fmt.Printf("num9对应的默认的类型为：%T",num5)

}
func main() {
	// complex	定义复数类型使用的关键字。
	var score complex64 = complex(1, 2)
	var number complex128 = complex(23.23, 11.11)
	fmt.Print("Score = ", score, " Number = ", number, "\n")
	// Score = (1+2i) Number = (23.23+11.11i)

	// 使用 real 获取复数的实部，使用 imag 获取复数的虚部。
	fmt.Print("Real Score = ", real(score), " Image Score = ", imag(score), "\n")
	// Real Score = 1 Image Score = 2
}
字符类型

Golang中没有专门的字符类型，如果要存储单个字符(字母)，一般使用 byte 来保存（Unicode 码）。

	var c1 byte = 'a'
	fmt.Printf("c1 的值：%d ,对应的具体的字符为：%c", c1, c1)
	// c1 的值：97 ,对应的具体的字符为：a

	//定义字符类型的数据：
	var c2 int16 = '中'
	fmt.Printf("c1 的值：%d ,对应的具体的字符为：%c", c2, c2)

	var int1 int = '中'
	fmt.Println(int1)

}

##### 数据类型转换

数值类型之间转换

Go 在不同类型的变量之间赋值时需要显式转换，并且只有显式转换(强制转换)。

语法：T(v) T : 数据类型 v : 需要转换的变量 将值 v 转换为类型 T

func main() {
	//进行类型转换：
	var n1 int = 100
	//var n2 float32 = n1  // 无法直接转换

	var n2 float32 = float32(n1)
	fmt.Println(n2)

	//将int64转为int8的时候，编译不会出错的，但是会数据的溢出
	var n3 int64 = 888888
	var n4 int8 = int8(n3)
	fmt.Println(n4) //56

	var n5 int32 = 12
	var n6 int64 = int64(n5) + 30 //一定要匹配=左右的数据类型
	fmt.Println(n6)

	var n7 int64 = 12
	var n8 int8 = int8(n7) + 127 //编译通过，但是结果可能会溢出
	//var n9 int8 = int8(n7) + 128 //编译不会通过
	fmt.Println(n8) // -117

}
数值类型转换为 string

方式1：fmt.Sprintf("%参数",表达式) 方式2：使用 strconv 包的函数
参数参考：https://studygolang.com/pkgdoc

func main() {
	var n1 int = 19
	var n2 float64 = 4.78
	var s1 string = fmt.Sprintf("%d", n1)
	fmt.Printf("s1对应的类型是：%T ，s1 = %#v \n", s1, s1)
	var s2 string = fmt.Sprintf("%f", n2)
	fmt.Printf("s2对应的类型是：%T ，s2 = %q \n", s2, s2)

	var n3 bool = true
	var s3 string = strconv.FormatBool(n3)
	fmt.Printf("s3对应的类型是：%T ，s3 = %q \n", s3, s3)
	var s4 string = strconv.FormatInt(int64(n1), 10) //参数：第二个参数指定字面值的进制形式为十进制
	fmt.Printf("s1对应的类型是：%T ，s1 = %q \n", s4, s4)
	var s5 string = strconv.FormatFloat(n2, 'f', 9, 64)
	//第二个参数：'f'（-ddd.dddd）  第三个参数：9 保留小数点后面9位  第四个参数：表示这个小数是float64类型
	fmt.Printf("s2对应的类型是：%T ，s2 = %q \n", s5, s5)
	// 4.780000000

}
string 类型转数值类型

方式：使用 strconv 包的函数

注意：string向基本数据类型转换的时候，一定要确保 string 类型能够转成有效的数据类型，否则最后得到的结果就是按照对应类型的默认值输出

func main() {
	//string-->bool
	var s1 string = "true"
	//ParseBool这个函数的返回值有两个：(value bool, err error)
	b, _ := strconv.ParseBool(s1)
	fmt.Printf("b的类型是：%T,b=%v \n", b, b)

	//string---》int
	var s2 string = "-19"
	num1, _ := strconv.ParseInt(s2, 10, 64)
	fmt.Printf("num1的类型是：%T,num1=%v \n", num1, num1)

	//string-->float32/float64
	var s3 string = "3.14"
	f1, _ := strconv.ParseFloat(s3, 64)
	fmt.Printf("f1的类型是：%T,f1=%v \n", f1, f1)
	
}

3.2 复杂数据类型

指针

基本数据类型都有对应的指针类型，形式为 *数据类型

基本数据类型和指针：

func main() {
	var age int = 18
	//&符号+变量 就可以获取这个变量内存的地址
	fmt.Println(&age) //0xc0000a2058
}

指针变量： 1. & 取内存地址 2. * 根据地址取值

func main() {
	var age int = 18

	//定义一个指针变量：
	//ptr 指针变量的名字
	//ptr对应的类型是：*int 是一个指针类型 （可以理解为 指向int类型的指针）
	//&age 就是一个地址，是 ptr 变量的具体的值
	var ptr *int = &age
	fmt.Println(ptr)	// 0xc0000a2058
	fmt.Println("ptr 存储空间的地址为：", &ptr) 
	// 获取指针指向的变量的值
	fmt.Printf("ptr指向的数值为：%v", *ptr) //ptr指向的数值为：18
}
}

通过指针改变指向值:

func main(){
	var num int = 10
	aa(num)
	fmt.Println(num)	// 10
	bb(&num)
	fmt.Println(num)	// 200
}

func aa(int2 int)  {
	int2 = 100
}

func bb(int2 *int)  {
	*int2 = 200
}
