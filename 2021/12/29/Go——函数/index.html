Go——函数

1. 函数

基本语法： func 函数名（形参列表) [ 返回值类型列表 ] { 　　执行语句.. 　　return　返回值列表 }

  1. 函数名
  • 驼峰命名
  • 首字母不能是数字
  • 首字母大写该函数可以被本包文件和其它包文件使用
  • 首学母小写只能被本包文件使用

  1. 形参列表： 形参可以是一个参数，可以是n个参数，可以是0个参数

  2. 返回值类型列表 如果有多个返回值，用 () 括起来；如果返回值类型只有一个，可以省略 ()；如果没有返回值，什么都不用写。在函数调用时，如果有返回值不想接收，可以用 _ 忽略。

  3. 值传递和引用传递 基本数据类型和数组默认都是值传递的，即进行值拷贝

  1. Golang 中函数不支持重载

  2. 支持可变参数 函数内将可变参数当做切片来处理

  1. 在 Go 中函数也是一种数据类型，可以赋值给一个变量，通过该变量可以对函数调用。
  1. 函数既然是一种数据类型，因此在 Go 中，函数可以作为形参。

9. 为了简化数据类型定义，Go 支持自定义数据类型

基本语法：type　自定义数据类型名　数据类型
  1. 支持对函数返回值命名

2. 包

  1. package 进行包的声明 建议：包的声明和所在的文件夹同名

  2. main 包是程序的入口包 main 函数一定要放在 main 包下，否则不能编译执行

  3. 包的引入 语法：import "包的路径"
    包名是从 $GOPATH/src/ 后开始的，使用 / 进行路径分隔。 【注：未开启 go modules 的情况下】

  4. 包名和文件夹的名字，可以不一样，建议一致

  5. 一个目录下只能声明一个 package，所以一个目录下不能有重复的函数

  6. 可以给包取别名，取别名后，原来的包名就不能使用了

4. 匿名函数

匿名函数使用方式：

  1. 在定义匿名函数时就直接调用，这种方式匿名函数只能调用一次

2. 将匿名函数赋给一个变量，再通过该变量来调用匿名函数

5. 闭包

闭包就是一个函数和与其相关的引用环境组合的一个整体

闭包的本质：闭包本质依旧是一个匿名函数，只是这个函数引入外界的变量/参数 匿名函数 + 引用的变量/参数 = 闭包

（ 当一个函数的返回值是另外一个函数，而返回的这个函数如果调用了其父函数内部的变量，且返回的这个函数在外部被执行，就产生了闭包。闭包是一个环境，具体指的就是外部函数--高阶函数。）

特点： 1. 返回的是一个匿名函数，但是这个匿名函数引用到函数外的变量/参数 ,因此这个匿名函数就和变量/参数形成一个整体，构成闭包。 2. 闭包中使用的变量/参数会一直保存在内存中

用处：

  1. 读取函数内部的变量
  2. 这些变量的值始终保持在内存中，不会在外层函数调用后被自动清除

优点： 1. 变量长期驻扎在内存中 2. 避免全局变量的污染

缺点： 1. 常驻内存 会增大内存的使用量

5. defer关键字

defer提供延迟机制，会将要延迟执行的方法“压栈”，当 defer 被触发时，将所有“压栈”的方法“出栈”并执行。

defer 的执行时机： 1. 包裹 defer 的函数返回时 2. 包裹 defer 的函数执行到末尾时 3. 所在的 goroutine 发生 panic 时

【注】调用 os.Exit() 方法退出程序时，不会执行

6. 系统函数

6.1 内置函数

new 函数： 分配内存，主要用来分配值类型（int系列, float系列, bool, string、数组和结构体 struct）

func new(Type) *Type 其第一个实参为类型，而非值。其返回值为指向该类型的新分配的零值的指针。

make函数: func make(Type, size IntegerType) Type

make分配并初始化一个类型为切片、映射、或通道的对象。其第一个实参为类型，而非值。make的返回类型与其参数相同，而非指向它的指针。

6.1 字符串相关函数

  1. 字符串转整数 n, err := strconv.Atoi("66")

  2. 整数转字符串 str := strconv.Itoa(6887)

  3. 不区分大小写比较字符串是否相同 func EqualFold(s, t string) bool

  4. 判断 s 是否有前缀字符串 prefix func HasPrefix(s, prefix string) bool

  5. 判断 s 是否有后缀字符串 suffix func HasSuffix(s, suffix string) bool

  6. 判断字符串 s 是否包含子串 substr func Contains(s, substr string) bool

  7. 返回字符串 s 中有几个不重复的 sep 子串 func Count(s, sep string) int

  8. 子串 sep 在字符串s中第一次出现的位置，不存在则返回-1 func Index(s, sep string) int

  9. 子串 sep 在字符串 s 中最后一次出现的位置，不存在则返回-1 func LastIndex(s, sep string) int

  10. 将所有字母都转为对应的小写 func ToLower(s string) string

  11. 将所有字母都转为对应的大写 func ToUpper(s string) string

  12. 返回 count 个 s 串联的字符串。 func Repeat(s string, count int) string

  13. 将 s 中前 n 个不重叠 old 子串都替换为 new 的新字符串 func Replace(s, old, new string, n int) string

  14. 将 s 前后端所有 cutset 去掉 func Trim(s string, cutset string) string

  15. 将 s 前后端所有空白都去掉 func TrimSpace(s string) string

  16. 用去掉 s 中出现的 sep 的方式进行分割，返回生成的所有片段组成的切片 func Split(s, sep string) []string 每一个 sep 都会进行一次切割，即使两个 sep 相邻，也会进行两次切割。
