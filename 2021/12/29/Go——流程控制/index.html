Go——流程控制

1. 分支结构

1.1 if 分支

基本语法：

if 条件表达式1 { 　逻辑代码1 } else if　条件表达式2 { 　逻辑代码2 } ....... else { 　逻辑代码n }

  • 条件表达式左右的 () 可以不写，也建议不写。
  • if 和表达式中间，一定要有空格。
  • {} 是必须有的，即使逻辑代码只有一行。
  • else 和上一个代码块结束的 { 必须在同一行，不能换行

1.2 switch 分支

基本语法：

switch 表达式 { 　case 值1,值2,.….: 　　　语句块1 　case 值3,值4,...: 　　　语句块2 　 .... 　default : 　　　 语句块 }

  1. switch 后是一个表达式（即:常量值、变量、一个有返回值的函数等）
  2. case 后面的值如果是常量值(字面量)，则要求不能重复

  1. case 后的各个值的数据类型，必须和 switch 的表达式数据类型一致

  2. case 后面可以带多个值，使用逗号间隔

  3. case 后面不需要带 break ，默认不穿透

  4. default 语句不是必须的，位置也是随意的 (所有 case 不匹配才执行)

  5. switch 后也可以不带表达式，当做 if 分支来使用

  6. 利用 fallthrough 关键字进行 switch 穿透，继续执行下一个case

java 中的 switch：

case 标签必须为字符串常量或字面量。 支持的类型： >Java5 以前，只支持 byte，short，char，int 类型 >Java5 引入了枚举类型和 byte，short，char，int 的包装类 >从Java7开始，支持 String 类型

（byte、short、char 类型可以在不损失精度的情况下向上转型成int类型。枚举类型的支持是因为枚举类有一个 ordinal 方法,该方法返回一个 int 类型的值。支持 String 是利用 String 的 hash 值，本质上也是 switch-int 结构，并且通过 equals 方法来防止 hash 冲突，用 switch-byte 结构精确匹配。)

2. 循环结构

Go 语言中只有 for 循环。

2.1 基本格式

for init; condition; post { } for condition { } for { }

2.2 循环控制关键字

控制语句 描述
break 经常用于中断当前 for 循环或跳出 switch 语句
continue 跳过当前循环的剩余语句，然后继续进行下一轮循环
goto 将控制转移到被标记的语句

以上关键字均支持标签。 （语法和 java 相同）

2.4 for range

for range 可以遍历数组、切片、字符串、map 及通道，for range 语法上类似于其它 java 中的 foreach 语句，一般形式为：

for key, val := range coll { 　　... }
