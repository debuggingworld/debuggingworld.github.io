idea——使用技巧

1. 代码模板

1.1 fori / forr / (int.fori/forr)

快速生成 for 循环

1.2 sout / soutv / sout / souf / soutm / serr

sout：快速生成 System.out.println();
soutv：输出当前上下文变量
soutm : 打印当前方法签名
soutp：打印当前方法参数
souf：格式化输出字符串
serr：prints a string to System.err

1.3 ifn / inn

ifn：判断上下文变量是否为空

inn：判断上下文变量是否不为空

1.4 surround with

快捷键 Ctrl + alt + T

选择 Surround with Callable：

2. 代码重构

  • extract
  • change signature
  • rename

2.1 extract（提取）

// method(M)、constant(C)、fields(F)、variables(V)、parameters(P)

  • 提取方法（Ctrl + Alt + M）

  • 提取常量（Ctrl + Alt + C）

  • 提取字段（Ctrl + Alt + F）

  • 提取变量（Ctrl + Alt + V）

  • 提取参数（Ctrl + Alt + P）

2.2 change signature

快捷键：Ctrl+F6

2.3 rename

快捷键：shift+F6

也可以在按完快捷键之后再按 shift+F6，设置重命名范围。

3. 常用快捷键

显示快捷键插件：Presentation Assistant

  • alt+enter
    结合上下文显示 error 的 fix 信息

    代码补全

    lambda 表达式生成

  • F2

    快速定位到报错行，并且多次按 F2 可在不同报错位置切换。

  • alt + 1

    快速定位到项目窗口，定位到 project 窗口之后可以直接输入进行搜索，定位到文件后直接回车打开。也可以按 esc 光标回到编辑区。

  • esc
    关闭任何上层窗口，回到编辑区域。
  • Ctrl + e
    显示最近修改的文件

  • Ctrl+b / Ctrl+alt+b
    Ctrl+b：进入到类的实现
    Ctrl+alt+b：进入到接口的实现类

  • alt + F7
    定位类、方法使用到的位置

  • Ctrl + Ctrl
    运行指令

  • Ctrl+W / Ctrl+shift+W
    选取代码块

  • Ctrl+/ 、Ctrl+shift+/
    Ctrl+/：单行注释
    Ctrl+shift+/：多行注释

  • Ctrl + shift + enter
    补全声明

  • Ctrl + alt + L
    格式化代码
  • Ctrl + shift + alt + T
    重构代码
    呼出面板后按选项前面的数字选择

  • Ctrl + shift + A
    查找操作

  • shift + shift
    Search everywhere
    可通过 tab 进行切换查找内容
  • Ctrl + D
    重复当前行或选择块

  • Ctrl+← / Ctrl + →
    光标按单词左右移动

  • alt+ ← / alt +→
    左右切换编辑区 table 页

  • alt+↑ / alt + ↓
    光标定位到上一个/下一个方法

  • shift + alt + 鼠标左键
    多行编辑
