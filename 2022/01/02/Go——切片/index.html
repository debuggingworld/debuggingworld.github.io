Go——切片

切片（slice）是对数组的一个连续片段的引用，所以切片是一个引用类型，这个片段可以是整个数组，也可以是由起始和终止索引标识的一些项的子集，需要注意的是，终止索引标识的项不包括在切片内。

一个 slice 由三个部分构成：指针、长度和容量。指针指向第一个 slice 元素对应的底层数组元素的地址，要注意的是 slice 的第一个元素并不一定就是数组的第一个元素。

1. 切片的定义

  1. 从数组或切片生成新的切片

    语法：slice [开始位置 : 结束位置] > 从数组或切片生成新的切片拥有如下特性： > + 取出的元素数量为：结束位置 - 开始位置； > + 取出元素不包含结束位置对应的索引，切片最后一个元素使用 slice[len(slice)] 获取； > + 当缺省开始位置时，表示从连续区域开头到结束位置； > + 当缺省结束位置时，表示从开始位置到整个连续区域末尾； > + 两者同时缺省时，与切片本身等效； > + 两者同时为 0 时，等效于空切片，一般用于切片复位。

       func main() {
	/* 数组长度为 5 */
	var  array = [5]int {1000, 2, 3, 17, 50}

	slice := array[0:4]
	// [1000 2 3 17]
	slice2 := slice[0:3]
	// [1000 2 3]

	fmt.Printf("%T",slice)
	// []int
	fmt.Printf("%v,%v",len(slice2),cap(slice2))
	// 3,5
}

  2. 使用 make() 函数构造切片

语法：make( []Type, size, cap )

make 底层创建一个数组，对外不可见

 
func main() {

	slice := make([]int, 5, 10)
	fmt.Println(slice)
	//[0 0 0 0 0]

	fmt.Printf("长度：%v，容量：%v", len(slice), cap(slice))
	// 长度：5，容量：10
}

  1. 直接声明新的切片
func main() {

	slice := []int{23, 45, 67}
	fmt.Println(slice)
	//[23 45 67]

	fmt.Printf("长度：%v，容量：%v", len(slice), cap(slice))
	// 长度：3，容量：3
}

2. 切片的遍历

方式1：for循环常规方式遍历 方式2：for-range 结构遍历切片

func main() {

	slice := []int{23, 45, 67}

	//方式1：普通for循环
	for i := 0; i < len(slice); i++ {
		fmt.Printf("slice[%v] = %v \t", i, slice[i])
	}
	fmt.Println("\n------------------------------")
	//方式2：for-range循环：
	for i, v := range slice {
		fmt.Printf("下标：%v ，元素：%v\n", i, v)
	}
}

3. 切片的操作

3.1 添加元素

用 append() 函数为切片动态添加元素时，如果空间不足，切片就会进行“扩容”。切片在扩容时，容量的扩展规律是按容量的 2 倍数进行扩充。

func main() {
	var a []int
	a = append(a, 1)                 // 追加1个元素
	a = append(a, 1, 2, 3)           // 追加多个元素, 手写解包方式
	a = append(a, []int{1, 2, 3}...) // 追加一个切片, 切片需要解包

	fmt.Println(a)
	// [1 1 2 3 1 2 3]

	var b = []int{1, 2, 3}
	b = append([]int{0}, b...)          // 在开头添加1个元素
	b = append([]int{-3, -2, -1}, b...) // 在开头添加1个切片
	fmt.Println(b)
	// [-3 -2 -1 0 1 2 3]

	var c []int = []int{1, 2, 3}
	c = append(c[:1], append([]int{100}, c[1:]...)...)     // 在第i个位置插入x
	c = append(c[:2], append([]int{0, 0, 0}, c[2:]...)...) // 在第i个位置插入切片
	fmt.Println(c)
	// [1 100 0 0 0 2 3]
}

在切片开头添加元素一般都会导致内存的重新分配，而且会导致已有元素全部被复制 1 次，因此，从切片的开头添加元素的性能要比从尾部追加元素的性能差很多。

3.2 切片复制

语法：copy( destSlice, srcSlice []T) int 将 srcSlice 复制到 destSlice

copy() 可以将一个数组切片复制到另一个数组切片中，如果加入的两个数组切片不一样大，就会按照其中较小的那个数组切片的元素个数进行复制。

来源和目标的类型必须一致，copy() 函数的返回值表示实际发生复制的元素个数。

func main() {
	slice1 := []int{1, 2, 3, 4, 5}
	slice2 := []int{5, 4, 3}
	copy(slice2, slice1) // 只会复制slice1的前3个元素到slice2中
	fmt.Println(slice2)
	// [1 2 3]

	copy(slice1, slice2) // 只会复制slice2的3个元素到slice1的前3个位置
	fmt.Println(slice1)
	// [1 2 3 4 5]
}

3.3 删除元素

func main() {
	a := []int{1, 2, 3, 4, 5, 6, 7, 8, 9}
	a = a[2:] // 删除开头N个元素
	a = append(a[:0], a[3:]...) // 删除开头N个元素

	fmt.Println(a)
	// [6 7 8 9]

	a = append(a[:1], a[1+1:]...) // 删除中间N个元素
	
	a = a[:len(a)-2] // 删除尾部N个元素
	
	fmt.Println(a)
	// [6]
}
