Go——Map

1. 定义 Map

var　map_variable　map[key_data_type]　value_data_type

var student map[int]string
  • key、value的类型：bool、数字、string、指针、channel 、还可以是只包含前面几个类型的接口、结构体、数组
  • key 通常为 int 、string 类型，value 通常为数字（整数、浮点数）、string、map、结构体
  • key：slice、map、function 不可以
  • map 只申明不会进行内存分配
  • map 的 key-value 是无序的
  • key 是不可以重复的
  • value 可以重复

2. 初始化

func main() {

	var student map[int]string
	// 第一种
	student = make(map[int]string, 10)
	student[00001] = "张三"

	// 第二种
	stu := make(map[int]string)
	stu[00002] = "李四"

	// 第三种
	stud := map[int]string{
		00003: "王五",
		00004: "赵六",
	}

	fmt.Println(stud)
}

3. 操作

  1. 增加、更新操作 map["key"] = value 如果 key 不存在，进行新增，如果 key 存则覆盖。

  2. 删除操作 delete(map，"key") 如果 key 存在，就删除该 key-value，如果 key 不存在，不操作也不会报错。

  3. 清空操作 （1）可以遍历key，逐个删除 （2）或者map = make(...)，make一个新的，让原来的成为垃圾，被gc回收

  4. 查找操作 value ,bool = map[key] value 为返回的 value，bool 为是否返回

  5. 获取长度：len函数

  6. 遍历：for-range

func main() {

	student := make(map[int]string)

	student[00001] = "张三"
	student[00001] = "李四"
	student[00002] = "王五"
	fmt.Println(student)
	// map[1:李四 2:王五]

	// 查找
	s := student[0001]
	fmt.Println(s) // 李四
	s2, b := student[0004]
	fmt.Println(s2, b) // false

	for i, s3 := range student {
		fmt.Printf("%v --- %v \n", i, s3)
	}

	// 删除
	delete(student, 001)
	delete(student, 003)
	fmt.Println(len(student)) // 1

	// 清空
	student = make(map[int]string)
	fmt.Println(len(student)) // 0

}

4. 比较

map 之间也不能进行相等比较；唯一的例外是和nil进行比较。要判断两个 map 是否包含相同的 key 和 value，我们必须通过一个循环实现：

func equal(x, y map[string]int) bool {
	if len(x) != len(y) {
		return false
	}
	for k, xv := range x {
		if yv, ok := y[k]; !ok || yv != xv {
			return false
		}
	}
	return true
}
